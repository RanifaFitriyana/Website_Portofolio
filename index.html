Beberapa bahasa pemrograman yang sudah pernah saya gunakan yaitu PHP, Java, Python, Javascript, dan Typescript. Oleh karena itu, saat ini saya sedang mempelajari web development lebih lanjut karena tujuan saya ingin menjadi seorang Web Developer untuk kedepannya.
